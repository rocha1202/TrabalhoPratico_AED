# Temas
Os temas são paleta de cores que podem ser são usuados pelo programa.
## Tema de configuração padrão
O tema de configuração padrão se encontra na pasta de configuração global com o nome de [theme.config](../.config/.theme.config). Este arquivo não deve permanecer inalterado.
## Temas customizados dos usuarios
Cada usuario pode criar o seu tema na sua pasta com o nome de _theme.config_.
Com o arquivo criar é só subscrever as cores desejadas.
Os atributos validos são:
- PRINCIPAL
- SECUNDARIA
- ESCURO
- CLARO
- BRANCO
